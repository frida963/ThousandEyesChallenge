Dentro de mi folder creo lo siguiente:
    carpeta "docs"
    carpeta "tests"
    readme.md
    main.py 

creo un ambiente virtual con venv
     python -m venv venv-api
(con control shft p puedo elegir el ambiente si ya lo tengo creado)

activo el ambiente virtual
    source venv-api/bin/activate

instalo lo que necesito
    pip install wheel
    pip install fastapi
    pip install  --upgrade pip
    pip install "uvicorn[standard]"
    pip install sqlalchemy
    pip install psycopg2-binary
envio esto a un archivo con requerimeintos 
     pip freeze > ./requirements.txt

para correr el codigo utilizo uvicorn
    uvicorn main:app
(es el nombre del .py : el del modulo)

toda peticion tiene le siguiente formato
    protocolo://servidor:puerto/carpeta/recurso

Tambien puedo crear el launch.json para correr el codigo
esto es en vscode en la parte de run y debug










para desactivar el ambiente 
    deactivate
